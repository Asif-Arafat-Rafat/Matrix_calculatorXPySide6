Matrix Calculator 🧮 (PySide6 GUI)

➖ Matrix Subtraction

✖️ Matrix Multiplication

🧩 Determinant Calculation

🧮 Matrix Transpose

🔄 Inverse Calculation (if applicable)

✅ Clean and responsive PySide6 GUI
